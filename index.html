Crystal Countdown
Rules
×

Goal: To reduce the target number shown on the top crystal to exactly 0.

Every crystal has a random value hidden in it. As you click any of the four crystals, the value hidden in it is subtracted from the target number. If the target number is reduced to a number lower than 0, you lose. If the target number is reduced to exactly 0, you win.
